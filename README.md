# jogo de foguete

Space Rocket Game é um pequeno jogo desenvolvido utilizando HTML, CSS e JavaScript. No jogo, o jogador controla um foguete que se movimenta pelo espaço enquanto precisa desviar de meteoros que surgem na tela.

## Sobre o jogo

A ideia do jogo é simples: manter o foguete em movimento e evitar colisões com os meteoros pelo maior tempo possível. Conforme o jogo acontece, o jogador precisa reagir rapidamente para continuar sobrevivendo.

## Como jogar

Use as teclas A e D do teclado para mover o foguete pela tela e desviar dos meteoros. O objetivo é evitar qualquer colisão e permanecer no jogo pelo maior tempo possível.

## Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript

## Objetivo do projeto

Este projeto foi criado com o objetivo de praticar conceitos básicos de desenvolvimento web e lógica de programação, incluindo:

* manipulação de elementos no DOM
* movimentação de objetos usando teclado
* detecção de colisões
* criação de animações no navegador
